## **Run Locally**

Clone the project

```bash
  git clone https://github.com/omgismartinez/rastreadito.git
```

Go to the project directory

```bash
  cd rastreadito
```

Install dependencies

```bash
  npm install | yarn
```

Start the server

```bash
  npm run dev | yarn dev
```
